Data Splitting — Important Techniques

A compact list formatted for readability with Calibri body text.

  • Train-Test Split — Basic single split (e.g., 80/20).
  • Train-Validation-Test — Three-way split for tuning and final evaluation.
  • K-Fold Cross Validation — Robust averaging across k folds.
  • Stratified K-Fold — Keeps class proportions per fold.
  • Repeated K-Fold — Multiple runs for variance reduction.
  • Leave-One-Out (LOOCV) — Extreme k-fold with k = n.
  • Group K-Fold — Keeps groups (e.g., users) intact across splits.
  • Nested CV — For hyperparameter selection and unbiased evaluation.
  • TimeSeriesSplit — Respect temporal order for time-series models.
  • Rolling Window & Expanding Window — Sliding / growing training windows.
  • Shuffle Split / Monte Carlo CV — Random repeated splits.
  • Bootstrap Sampling — Sampling with replacement for uncertainty estimates.
  • Purged K-Fold — Purges overlap to prevent look-ahead bias (finance).
